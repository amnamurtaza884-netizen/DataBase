# PHP Student Table Project

This project is a **PHP-based student table application** that demonstrates database creation, table generation, record insertion, and styled data display using MySQL & CSS.

---

## Project Overview

The project performs the following tasks:

- Creates a MySQL database (`lab_db`) if it does not exist
- Creates a table `students` with 9 fields (columns)
- Inserts 5 different student records
- Displays all records in a **styled HTML table**
- Prevents duplicate records by recreating the table on each run
- Includes professional CSS styling for a clean and readable layout

---

## Table Structure

The `students` table contains the following fields:

1. ID (Primary Key, Auto Increment)  
2. Full Name  
3. Email  
4. Phone  
5. Gender  
6. Course  
7. City  
8. Age  
9. Registration Date  

---

## Features

- Database and table creation via PHP  
- Multiple record insertion  
- Dynamic record fetching and display  
- Styled table with hover effects and alternating row colors  
- Professional page layout with background and footer  
- All operations in one PHP file (`db.php`)  
- No external libraries used

## How to Run

1. Install **Laragon** or **XAMPP**  
2. Place `db.php` in the `www` folder  
3. Start Apache and MySQL servers  
4. Open in browser:
http://localhost/test/db.php
